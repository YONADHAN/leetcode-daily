# Day 3 - 2026-05-16

## Problems Solved

1. 69. Sqrt(x) (Easy)

## Approach Summary

* Used **brute force iteration**
* Started loop from `i = 1`
* Continued while `i * i <= x`
* Stored last valid value in `z`
* If `i * i === x`, returned immediately (perfect square)
* Returned `z` for non-perfect squares (floor value)

## Key Learnings

* Understood how to find **square root without using `Math.sqrt()`**
* Learned to track **last valid answer (floor value)**
* Improved understanding of **loop termination condition (`i * i <= x`)**
* Recognized inefficiency of brute force for large inputs
* Learned that this problem can be optimized using **Binary Search**
