## Problems Solved

1. 125. Valid Palindrome (Easy)

## Approach Summary

* Used **two-pointer technique** (`left` and `right`)
* Traversed string from both ends
* Skipped non-alphanumeric characters using helper function (`charCodeAt`)
* Compared characters after converting to lowercase
* Moved pointers inward after each comparison
* Returned `false` on mismatch, else `true`

* Improved understanding of **two-pointer pattern for strings**
* Fixed common bugs:

  * Infinite loop due to missing pointer movement
  * Case sensitivity issues
  * Edge cases with special characters
* Learned why `typeof` does NOT work for character validation
