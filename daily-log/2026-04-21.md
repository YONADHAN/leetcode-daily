# Day 4 - 2026-05-17

## Problems Solved

1. 290. Word Pattern (Easy)

## Approach Summary

* Split string `s` into words using `split(' ')`
* Checked if `pattern.length !== words.length` → return `false`
* Used an object to map **pattern character → word**
* Iterated through pattern:

  * If character not mapped:

    * Checked if word already exists using `Object.values().includes()`
    * If yes → return `false` (violates one-to-one mapping)
    * Else → store mapping
  * If already mapped:

    * Compared mapped word with current word
    * If mismatch → return `false`
* Returned `true` if all mappings are valid

## Key Learnings

* Learned concept of **bijection (one-to-one mapping)**
* Understood why **duplicate value mapping must be avoided**
* Practiced using **objects as hash maps**
* Realized that `Object.values().includes()` is inefficient (O(n))
* Learned better approach using **two hash maps (char→word & word→char)**
