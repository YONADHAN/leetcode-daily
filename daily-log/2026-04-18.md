# Day 2 - 2026-05-16

## Problems Solved

1. 205. Isomorphic Strings (Easy)

## Approach Summary

* Used object (hash map) to store character mapping
* Traversed both strings using a loop
* Checked if character already mapped:

  * If yes → validate mapping
  * If no → ensure no duplicate mapping using `Object.values().includes()`
* Ensured one-to-one mapping (bijection)

## Key Learnings

* Understood concept of **one-to-one character mapping**
* Learned that mapping must be **bijective (both unique and consistent)**
* Realized performance issue with `Object.values().includes()`
* Got clarity on how hash maps help in string problems
